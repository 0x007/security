## Wargame In Linux 
[io.smashthestack](http://io.smashthestack.org/)是专门针对Linux系统安全从业人员的培训整套练习环境。
并且针对不同领域方向，都有相关的关卡，玩家必须想尽办法得到下一关的密码。
